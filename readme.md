# Rollout Calculator (A12 / A12X)

A lightweight, mobile-friendly rollout calculator for 1/12 scale on-road cars.
Includes:

- Live rollout calculation
- A12 / A12X legal gear range checking
- Target rollout gear finder
- Clean, modern UI
- 100% client-side (no backend)
